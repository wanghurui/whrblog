小米路由器3刷入LEDE

发表于2019-01-13更新于2021-08-21

不得不吐槽小米官方的固件，真的是要什么没什么，一个残缺的opkg包管理已经不能满足我折腾的欲望了，一个深度修改阉割版的openwrt有什么好玩的。 近日在恩山论坛上看到了有人编译好了的LEDE，感谢@ptpt52大神提供编译。

刷机有风险，如果个人刷机出现了问题与我，ptpt52，恩山无关，自重

准备

通过小米官方的方法，开启ssh，root方法1（http://www.miui.com/thread-4529081-1-1.html 方法2（https://jingyan.baidu.com/article/a24b33cd1080f719fe002b8e.html） 固件下载地址https://downloads.x-wrt.com/rom/ 用putty 登陆ssh，想办法把固件传到设备上 /tmp 目录，或者用u盘 刷机需要的固件文件两个 x-wrt-*.*-*********-ramips-mt7620nand-xiaomi_miwifi-r3-squashfs-kernel1.bin x-wrt-*.*-*********-ramips-mt7620nand-xiaomi_miwifi-r3-squashfs-rootfs0.bin

正式刷机

 

nvram set flag_last_success=1
nvram set boot_wait=on
nvram set uart_en=1
nvram commit

 

mtd write xxxxx-kernel1.bin kernel1
mtd write xxxxx-rootfs0.bin rootfs0
reboot

注意前面4条命令非常重要，它开启串口，因为小米默认锁死串口，如果你不开启，万一刷机失败或者出现意外，再也救不回来了，一定要网线刷机，我用wifi已经刷死过一次了

刷机完成

固件无线默认名称：X-WRT_XXXX，密码：88888888 固件管理界面：http://192.168.15.1/ 管理界面账户密码：root/admin 主页：https://x-wrt.com/ 开发版本下载：https://downloads.x-wrt.com/rom/ SDK地址：https://downloads.x-wrt.com/rom/sdk/
