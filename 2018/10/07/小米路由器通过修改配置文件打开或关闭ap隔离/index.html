小米路由器通过修改配置文件打开或关闭AP隔离

发表于2018-10-07更新于2021-08-21

某天我ssh到我的香橙派上去，可是发现putty报错说connect timeout……. 百度无果，经过一次无意间的测试，发现问题出现在/etc/config/firewall上

config zone
option name ‘wan’
option network ‘wan’
option input ‘ACCEPT’
option output ‘ACCEPT’
option forward ‘ACCEPT’
option masq ‘1’ #就是这一行1代表关闭ap隔离&0代表打开
option mtu_fix ‘0’

最后一步/etc/init.d/firewall restart 或者你可以reboot
