小米路由器3 opkg

Created2018-04-02Updated2021-08-21

1.0安装opkg

复制opkg到小米路由的/data路径下

http://whrblog.online/viewfile.php?file_id=8 下载opkg

1.1换源

修改/etc/opkg.conf文件，替换为以下内容 可以直接在这里下载http://whrblog.online/viewfile.php?file_id=10 然后用winscp替换

1.2添加环境变量

添加环境变量。修改**/etc/profile文件，修改export PATH**为 export PATH=/bin:/sbin:/usr/bin:/usr/sbin:/data:/data/usr/sbin:/data/usr/bin 再这一行下面新添加一行 export LD_LIBRARY_PATH=/data/usr/lib 重启生效

2.0安装libc库

如果提示错误，缺少libc*。这里需进行安装，注意保持libc为最新版本。 cd /data wget http://openwrt.sangbo.pub/barrier_breaker/14.07/ramips/mt7620a/packages/base/libc_0.9.33.2-1_ramips_24kec.ipk opkg install libc_*.ipk rm -rf libc_0.9.33.2-1_ramips_24kec.ipk

成功
